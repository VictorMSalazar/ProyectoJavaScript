Proyecto: Compra online de Café Kawaii.
Mi proyecto es la pagina web de una cafetería (Café Kawaii en este caso), donde se verán las compras online para efectuar delivery o retiro en local.
La persona debe seleccionar su pedido, indicar la cantidad de productos y sobre $15.000 pesos de compra tendrán un dcto de 5%, y si incorporan un código, tendrán un 10% de descuento.
